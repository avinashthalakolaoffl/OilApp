OilApp is a full-stack MERN application for ordering oil online.

## Features
- User Registration & Login
- Place Oil Orders
- View Orders
- Update Order Status
- JWT Authentication
- MongoDB Database

## Tech Stack
- Frontend: React
- Backend: Node.js, Express
- Database: MongoDB Atlas
- Hosting: Render
